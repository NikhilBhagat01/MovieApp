# MovieApp

This is a Movie app where you can search and download movies available on the internet....

## Run Locally

Clone the project

```bash
  git clone https://github.com/NikhilBhagat01/MovieApp.git
```

Go to the project directory

```bash
  cd MovieApp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
